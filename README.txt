# Purpose:
Pandora for Youtube (Continuous playlist generator given an initial seed video, with ranking and like/dislike algorithms.)

# Technologies:
Django, Youtube-Python API, Youtube-Javascript API, JQuery, AJAX, Bootstrap

A. Youtube-Python API used to compile a ranked list of similar videos
B. Youtube-Javascript API used for displaying, playing and loading videos
C. JQuery used to make AJAX calls to communicate between Python and Javascript
D. Bootstrap used to construct a neat GUI
E. Django used as a framework to tie together all the technologies
