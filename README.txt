# Purpose:
Pandora for Youtube (Continuous playlist generator given an initial seed video, with ranking and like/dislike algorithms.)

# Technologies:
Django, Youtube-Python API, Youtube-Javascript API, JQuery, AJAX

1. Youtube-Python API used to compile a ranked list of similar videos
2. Youtube-Javascript API used for displaying, playing and loading videos
3. JQuery used to make AJAX calls to communicate between Python and Javascript
4. Django used as a framework to tie together all the technologies
